# 3. 방법론(파이프라인)

- image_size = 512, patch_size = 16
- token_dim = 768
- heatmap_size = 64
- heatmap_loss_fn = bce
- loss weights: lambda_hm=100, lambda_ang=3, lambda_lbl=1
- alignment weights: lambda_reason=0.5, lambda_obj=0.5

## Notation

- $B$ : batch size
- $N$ : number of people tokens before target-person selection
- $H, W$ : image size ($H = W = 512$)
- $P$ : patch size ($P = 16$)
- $T$ = $(H / P) \times (W / P)$ : number of image patch tokens ($T=1024$)
- $S$ = $\sqrt T$ : spatial token grid size ($S=32$)
- $d$ : decoder token dim ($d=768$)
- $m_b \in \{ 0, 1\}$ : in/out mask for batch sample $b$ (1 means in-frame)

## 1) Image Encoder + Projection

Image Encoder $\phi$ : pre-trained DINOv3-ViT-L/16 (frozen)

Projection Layer $\text{Proj}_V$

Image : $I \in \mathbb R^{3 \times H \times W}$

- $X = \phi(I) \in \mathbb R^{(1 + R + T) \times D_V}$
- $X^{proj} = \text{Proj}_V(X[(1+R) : ]) \in \mathbb R ^{T \times d}$
- $X^{img} = \text{reshape}(X^{proj}) \in \mathbb R^{d \times S \times S}$

$H = W = 512,\space P = 16,\space T = 1024,\space S = 32$

## 2) Gaze Encoder

Head crops & head bboxes : $\{{h, b}\}, \quad h \in \mathbb R^{3 \times 224 \times 224}, \space b \in \mathbb R^4$

Resnet18 $f$

- $e = f(h)$
- $X^{gaze} = \text{MLP}(e) + W_bb \in \mathbb R ^d$
- $v = \text{normalize}(\text{MLP}_v(e)) \in \mathbb R^{2}$

## 3) Gaze Decoder

Gaze-to-Image Cross-Attn : $\psi_{g\rightarrow i}$

Image-to-Gaze Cross-Attn : $\psi_{i \rightarrow g}$

Two-way block(코드 기준):

$
\Psi(\cdot)
=
\psi_{i \rightarrow g}\!\Big(
\text{MLP}\big(\psi_{g \rightarrow i}(\cdot)\big)
\Big)
$

- not-a-person token : $q_{na} \in \mathbb R^d$
- gaze token : $T^{gaze}_0 = \text{concat}(X^{gaze}, q_{na}) \in \mathbb R^{2 \times d}$
- image token : $T^{img}_0 = \text{flatten}(X^{img}) \in \mathbb R^{T \times d}$

- $(T^{gaze}_{l+1}, T^{img}_{l+1}) = \Psi(T^{gaze}_l, T^{img}_l), \quad l = 0, 1, ..., L-1$

- $T^{img} = T^{img}_L$
- $T^{gaze} = \text{LN}\!\left(T^{gaze}_L + \psi_{g\rightarrow i}(T^{gaze}_L, T^{img}_L)\right)$

여기서 $\psi_{g\rightarrow i}(q,k)$는 코드의 final attention(`q=gaze`, `k=v=image`)에 해당.

## 4) Heatmap & Label Branch

Heatmap : 

- $\tilde{X}^{img} = \text{Upscale}(T^{img}) \in \mathbb R^{d' \times H_{u} \times W_{u}}, \quad d' = d/8$
- 사람 토큰만 사용: $T^{gaze}_{person} = T^{gaze}[:-1]$
- $\tilde{X}^{gaze} = \text{MLP}_{hm}(T^{gaze}_{person}) \in \mathbb R^{N \times d'}$
- $\hat{H}^{raw} = \tilde{X}^{gaze}\left(\text{reshape}(\tilde{X}^{img}, d', H_uW_u)\right) \in \mathbb R^{N \times H_uW_u}$
- $\hat{H} = \text{reshape}(\hat{H}^{raw}, N, H_u, W_u) \xrightarrow[]{\text{if needed resize}} \mathbb R^{N \times 64 \times 64}$

Label embedding : 

- $\hat{z} = \text{normalize}(\text{MLP}_{label}(T^{gaze}_{person})) \in \mathbb R^{N \times d_{lbl}}$

Gaze vector (for angular loss) : 

- $\hat{v} = v \in \mathbb R^{N \times 2}$
